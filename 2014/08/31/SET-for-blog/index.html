在新电脑上的博客配置

  1. 安装Git,http://msysgit.github.io/
  2. 安装Node.js,http://nodejs.org/
  3. 安装hexo 利用 npm 命令即可安装。（在任意位置点击鼠标右键，选择Git bash）npm install -g hexo
  4. 在你喜爱的文件夹下（如H:\hexo），执行以下指令(在H:\hexo内点击鼠标右键，选择Git bash)，hexo init,npm install,Hexo 即会自动在目标文件夹建立网站所需要的所有文件。
  5. 现在我们已经搭建起本地的hexo博客了，执行以下命令(在H:\hexo)，hexo generate,hexo server然后到浏览器输入localhost:4000看看。好了，至此，本地博客已经搭建起来了，只是本地哦，别人看不到的。下面，我们要部署到Github。
  6. 前期步骤详见链接1.
    设置用户信息（适用于https方式，ssh方式也需要）
    git config --global user.name "hyf"//用户名
    git config --global user.email "hyf278@126.com"//填写自己的邮箱
    会生成.gitconfig文件：内容是
    [user]
    name = hyf
    email = hyf278@126.com
  7. 如果使用：git@github.com:pureal/pureal.github.io.git,需要生成SSH，https://help.github.com/articles/generating-ssh-keys/ ：
    ssh-keygen -t rsa -C "hyf278@126.com"
    Add your SSH key to GitHub
    ssh -T git@github.com
    如果使用：https://github.com/pureal/pureal.github.io.git （推荐使用https）,
    git config --global credential.helper wincred
    会在.gitconfig文件中加入
    [credential]
    helper = wincred
    表示生成credentials文件，只需一次输入用户名和密码
  8. 至此设置完毕。hexo generate,hexo deploy,生成、发布博客
  9. 如果以上方法行不通，那么备份theme文件夹,source文件夹，_config.yml（需要定期备份这三个文件）, hexo init重新部署
