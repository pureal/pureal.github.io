    进入目录

    强制用系统默认方式打开

    Shift+回车

    文件名排序配置参数

    SortUpper=0
    以数字命名的文件，按照第一个数字的大小排序。比如以下例子：
    内部培训教程1.txt
    内部培训教程10.txt
    内部培训教程11.txt
    内部培训教程12.txt
    内部培训教程2.txt
    内部培训教程3.txt
    SortUpper=1
    数字排序与SortUpper=0一样。文件排序时，以下划线开头的文件排后面。
    SortUpper=2
    数字文件，以大小来排序，比如：
    内部培训教程1.txt
    内部培训教程2.txt
    内部培训教程3.txt
    内部培训教程9.txt
    内部培训教程10.txt
    内部培训教程11.txt
    内部培训教程12.txt
    内部培训教程20.txt
    这就是古意人推荐使用这个参数的原因！汉字排序，同SortUpper=1。
    SortUpper=3
    数字文件，同SortUpper=2。汉字排序，同SortUpper=0。

    slister也分32位和64位，用的时候重新安装
    TCtorrent也分32位和64位，用的时候重新安装
    WFX对于32位和64位好像无需更改

菜单的更改在LANGUAGE\WCMD_CHN.MNU中，调用的是usercmd.ini定义的外部命令

Wcmd_chn.ini
[em_file_exchangename]
button=wcmicons.dll,12
cmd=%COMMANDER_PATH%\user\file_exchangename.bat
param=%P%S2
menu=交换文件名称

file_exchangename.bat
Set file1=%~nx1
Set file2=%~nx2
Cd /d “%~dp1”
Ren “%file1%” “$TMP”
Ren “%file2%” “%file1%”
Ren “$TMP” “%file2%”
